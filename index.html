Your Mission - Add Your Card!

  1. Fork or clone this repository to your local machine.
  2. Create a new branch: git checkout -b add-your-name
  3. Open index.html and find the comment "ADD YOUR CARD BELOW THIS LINE".
  4. Copy the example card block and update it with your own name, initials, role, and a fun fact.
  5. Choose a gradient for your avatar - e.g. linear-gradient(135deg, #f093fb, #f5576c).
  6. Stage your changes: git add index.html
  7. Commit: git commit -m "feat: add [your name] to team page"
  8. Push: git push origin add-your-name
  9. Go to GitHub and open a Pull Request targeting the main branch.
  10. Once approved and merged, refresh this page - your card will appear!
